\subsection{alsa-utils}

=== alsa-utils
  \item ``` alsa-utils ``` is a repository of tools to interact with
    ALSA devices
  \item \url{https://github.com/alsa-project/alsa-utils}


=== Controls
  \item ``` alsamixer ``` provides a ``` ncurse ``` based graphical
    interface to modify sound cards controls.
  \item ``` amixer ``` is a command line tool to set controls.
    \item The ``` scontrols ``` and ``` controls ``` commands list the
      available controls.
    \item The ``` contents ``` commands list the
      available controls and shows their content.
    \item The ``` cset ``` and ``` sset ``` commands allows modifying
      the controls.
    \item The ``` cget ``` and ``` sget ``` commands show the content of
      a specific control.
  \item ``` alsactl ``` is a tool that can save the control values to a
    file and restore them from a file.


=== Playback and capture
  \item ``` speaker-test ``` can generate tones or noises to play on
    specific channels with a specified rate.
  \item ``` aplay ``` plays an audio file. It is able to set many
    stream parameters.
  \item ``` arecord ``` can record an audio stream to a file.
